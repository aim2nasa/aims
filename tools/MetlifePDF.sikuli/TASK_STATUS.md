# MetLife 고객목록 클릭 자동화 - 작업 현황

## 목적

MetLife 고객목록 페이지에서 **모든 고객 행을 정확하게 클릭**하는 SikuliX GUI 자동화 스크립트 완성

### 핵심 요구사항
1. **행의 중앙을 정확하게 클릭** - 행 내 어딘가가 아닌 정확한 중앙
2. **모든 행 처리** - 초성 "ㄴ" 기준 총 27행 (13+13+1)
3. **두 가지 모드 지원**
   - Offset 모드: `-- ㄴ` (픽셀 계산 기반)
   - Arrow-nav 모드: `-- ㄴ --arrow-nav` (키보드 화살표 기반)

---

## 배경

### 환경 변경
- 브라우저 줌 **150% → 100%** 변경
- 이에 따라 SikuliX 이미지 파일 100% 줌 버전으로 교체
- 화면당 표시 행 수: **13행** (100% 줌 기준)

### 페이지 구성 (초성 "ㄴ")
| 페이지 | 행 수 | 고객 범위 |
|--------|-------|-----------|
| P1 | 13행 | 나루에스앤에프 ~ 남수연 |
| P2 | 13행 | 남원석 ~ 노휘소 |
| P3 | 1행 | 노휘안 |
| **합계** | **27행** | |

### 핵심 파라미터
```python
FIRST_ROW_OFFSET = 41           # 첫 페이지: 헤더 → 첫 행 중앙 (픽셀)
FIRST_ROW_OFFSET_SCROLLED = 32  # 스크롤 후 페이지: 헤더 → 첫 행 중앙
ROW_HEIGHT = 31                 # 행 간 간격 (픽셀)
ROWS_PER_PAGE = 13              # 화면당 표시 행 수
```
