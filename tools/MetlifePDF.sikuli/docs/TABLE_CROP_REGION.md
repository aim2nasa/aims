# 테이블 크롭 영역 정의

## 화면 해상도
- 전체 화면: **1920 x 1080**

## 크롭 좌표

| 항목 | 값 | 설명 |
|------|-----|------|
| X | 30 | 테이블 시작 X |
| Y | 386 | 테이블 시작 Y (헤더 포함) |
| WIDTH | 1880 | 테이블 너비 (1920 - 30 - 10) |
| HEIGHT | 580 | 테이블 높이 (15행 + 헤더 + 여유) |

## 크롭 영역 시각화

```
+------------------------------------------+ (0, 0)
|  MetLife 헤더/메뉴                        |
|------------------------------------------|
|  필터 영역 (고객명, 구분, 성별...)          |  <- 제외
|  초성 버튼 (ㄱ ㄴ ㄷ ... 전체 고객등록)      |  <- 제외
|------------------------------------------|
|  +------------------------------------+  | (30, 386)
|  | 고객명 | 구분 | 생년월일 | ...      |  |  <- 크롭 시작
|  |------------------------------------|  |
|  | 홍길동 | 계약 | 1990-01-01 | ...   |  |
|  | ...    | ...  | ...        | ...   |  |
|  | (15행) | ...  | ...        | ...   |  |
|  +------------------------------------+  | (1910, 966)
|                                          |
+------------------------------------------+ (1920, 1080)
```
