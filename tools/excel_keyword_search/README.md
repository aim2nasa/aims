# 📊 엑셀 키워드 검색 도구

엑셀 파일의 모든 탭에서 키워드를 검색하고, 일치하는 행의 전체 데이터를 표시하는 도구입니다.

## 🎯 주요 기능

- ✅ **모든 탭 검색**: 엑셀 파일의 모든 시트를 자동으로 검색
- ✅ **전체 데이터 표시**: 키워드가 포함된 행의 모든 컬럼 데이터 출력
- ✅ **키워드 강조**: 키워드가 포함된 셀에 ⭐ 표시
- ✅ **정확한 위치**: 탭 이름과 엑셀 행 번호 표시
- ✅ **대소문자 무시**: 대소문자 구분 없이 검색
- ✅ **부분 일치**: 키워드가 포함된 모든 데이터 검색

## 📦 설치

### 필수 요구사항

```bash
pip install pandas openpyxl
```

## 🚀 사용 방법

### 1. GUI 버전 (추천)

가장 간편한 사용 방법입니다.

```bash
python excel_search_gui.py
```

#### 사용 순서:
1. **📂 파일 선택** 버튼 클릭
2. 엑셀 파일 선택
3. 검색어 입력 (Enter로도 검색 가능)
4. **🔍 검색** 버튼 클릭
5. 결과 확인

![GUI 화면 예시]
- 파일 탐색기로 엑셀 선택
- 실시간 검색 결과 표시
- 스크롤 가능한 결과 영역
- 상태바에 진행 상황 표시

---

### 2. CLI 버전 (명령줄)

터미널에서 직접 실행하는 방법입니다.

#### 방법 1: 대화형 모드
```bash
python excel_search.py
```

실행 후 파일 경로와 검색어를 입력합니다.

#### 방법 2: 인자로 전달
```bash
python excel_search.py "파일경로.xlsx" "검색어"
```

**예시:**
```bash
python excel_search.py "D:\메트라이프전체고객리스트20240311.xlsx" "김철수"
```

## 💡 활용 사례

### 1. 고객 정보 통합 검색
여러 탭에 흩어진 동일 고객 정보를 한 번에 찾기
```bash
python excel_search.py "고객리스트.xlsx" "김철수"
```

### 2. 전화번호로 검색
```bash
python excel_search.py "고객리스트.xlsx" "010-1234-5678"
```

### 3. 주소로 검색
```bash
python excel_search.py "고객리스트.xlsx" "강남구"
```

### 4. 계약번호로 검색
```bash
python excel_search.py "고객리스트.xlsx" "ABC123"
```

## 🔧 기술적 특징

### 검색 알고리즘
- **전체 셀 검색**: 모든 컬럼의 모든 셀을 검색
- **부분 일치**: "김철" 검색 시 "김철수"도 찾음
- **대소문자 무시**: "ABC" 검색 시 "abc"도 찾음
- **빈 셀 무시**: NaN이나 빈 셀은 출력하지 않음

### 성능
- **백그라운드 검색** (GUI): UI가 멈추지 않음
- **효율적 메모리**: 매칭된 행만 메모리에 유지
- **병렬 처리 가능**: 여러 파일 동시 검색 가능

### 지원 형식
- `.xlsx` (Excel 2007+)
- `.xls` (Excel 97-2003)

## ⚠️ 주의사항

1. **대용량 파일**: 파일이 매우 크면 검색에 시간이 걸릴 수 있습니다
2. **병합 셀**: 병합된 셀은 첫 번째 셀의 값만 검색됩니다
3. **수식**: 수식의 결과값이 아닌 표시값을 검색합니다
4. **숨겨진 시트**: 숨겨진 시트도 검색됩니다

## 🛠️ 트러블슈팅

### "ModuleNotFoundError: No module named 'pandas'"
```bash
pip install pandas openpyxl
```

### "FileNotFoundError: [Errno 2] No such file or directory"
파일 경로를 큰따옴표로 감싸세요:
```bash
python excel_search.py "D:\파일명.xlsx" "검색어"
```

### GUI가 실행되지 않음 (Linux/Mac)
tkinter가 설치되어 있는지 확인:
```bash
# Ubuntu/Debian
sudo apt-get install python3-tk

# macOS
brew install python-tk
```
