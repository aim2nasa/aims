# SemanTree v0.7.1 - AIMS Document Viewer

AIMS MongoDB 문서 뷰어 및 시맨틱 트리 분석 도구

## 주요 기능

### 1. 문서 트리 뷰
- **태그별 분류**: 문서를 태그로 그룹화하여 트리 형태로 표시
- **연도/월별 분류**: 업로드 날짜 기준으로 연도별/월별 그룹화
- **검색 및 필터**: 파일명, 태그, 날짜 범위로 문서 검색
- **다중 선택 모드**: AND/OR 조건으로 여러 태그 선택 가능

### 2. 문서 목록
- 선택한 태그/날짜의 문서 목록 표시
- 파일명, 업로드 날짜, 태그 정보 표시
- 더블클릭으로 문서 상세 보기

### 3. Raw 데이터 뷰어
- **DB/Collection 선택**: MongoDB의 모든 DB와 Collection 탐색
- **문서 네비게이션**: 이전/다음 버튼으로 문서 이동
- **요약/전체 보기**: JSON 데이터 요약 또는 전체 표시 토글
- **클립보드 복사**: Raw JSON 데이터 복사
- **실시간 새로고침**: 현재 문서를 DB에서 자동으로 다시 조회 (3초 간격, 기본 활성화)
- **수동 새로고침**: 🔃 버튼으로 현재 문서만 즉시 새로고침
- **자동 새로고침 토글**: 🔄 버튼으로 자동 새로고침 켜기/끄기
- **패턴 기반 삭제**: 🗑️ 버튼으로 필드 패턴에 맞는 문서 일괄 삭제 (모든 컬렉션)
- **텍스트 검색** ✨ NEW: 컬렉션 내 특정 문구가 포함된 문서 검색 및 네비게이션

### 4. 태그 통계
- 전체 태그 빈도수 및 비율 통계
- 태그별 사용 현황 분석

## 시스템 요구사항

- Windows 10/11
- SSH 접속 가능한 환경 (tars.giize.com)
- MongoDB 접근 권한

## 설치 및 실행

### 방법 1: Windows 실행 파일 (권장) ⭐

1. `dist/SemanTree.exe` 다운로드
2. 더블클릭으로 실행
3. SSH 터널이 자동으로 연결됨 (백그라운드)

**특징:**
- ✅ Python 설치 불필요
- ✅ SSH 콘솔 창 숨김 처리
- ✅ 단일 파일로 배포 가능
- ✅ 파일 크기: 약 19MB

### 방법 2: Python 스크립트 실행

```bash
# 필수 패키지 설치
pip install pymongo tkinter

# 실행
python semantree.py
```

**참고**: SSH 터널이 자동으로 시작됩니다. tars.giize.com SSH 접속 권한이 필요합니다.

## EXE 빌드 방법

새로운 버전을 빌드하려면:

```bash
# Windows 배치 파일 사용
build_exe.bat

# 또는 직접 PyInstaller 사용
py -3 -m PyInstaller --name=SemanTree --onefile --windowed --noconfirm semantree.py
```

빌드 옵션:
- `--onefile`: 단일 실행 파일 생성
- `--windowed`: 콘솔 창 숨김 (GUI만 표시)
- `--noconfirm`: 기존 빌드 자동 덮어쓰기

## 사용 방법

### 1. 초기 연결
- 앱 시작 시 자동으로 SSH 터널 연결
- MongoDB 연결 성공 시 상단에 녹색 체크 표시

### 2. 문서 탐색
1. 좌측 트리에서 분류 기준 선택 (태그별/연도별/월별)
2. 태그 또는 날짜 노드 클릭
3. 우측에 해당 문서 목록 표시
4. 문서 더블클릭으로 상세 보기

### 3. Raw 데이터 탐색 ✨ NEW
1. "📋 Raw 데이터" 탭 선택
2. Database 드롭다운에서 DB 선택
3. Collection 드롭다운에서 컬렉션 선택
4. "🔄 로드" 버튼 클릭
5. 이전/다음 버튼으로 문서 탐색
6. "표시: 요약/전체" 버튼으로 JSON 표시 모드 전환
7. **실시간 모니터링**:
   - "🔄 자동 (3초)" 버튼: 자동 새로고침 활성화 (기본값)
   - "🔃 새로고침" 버튼: 현재 문서만 수동으로 즉시 새로고침
   - 외부에서 DB를 수정하면 3초 이내에 화면에 자동 반영됨
8. **패턴 기반 삭제** (모든 컬렉션):
   - "🗑️ 패턴 삭제" 버튼 클릭
   - 필드명과 패턴 입력 (예: upload.destPath, /tmp/test-*)
   - 와일드카드 지원: * (임의 문자열), ? (한 글자)
   - 대소문자 구분 옵션
   - 삭제 전 매칭 문서 개수 확인 및 안전 확인 다이얼로그
   - 컬렉션별 기본값 자동 설정:
     - customers: "personal_info.name", "테스트고객*"
     - files: "upload.destPath", "/tmp/test-*"
   - 마지막 설정값 자동 저장 및 복원

### 4. 검색 및 필터
- **검색**: 파일명 또는 태그로 검색
- **날짜 필터**: 시작일~종료일 범위 지정 (예: 2024-01-01)
- **다중 선택**: Ctrl 키로 여러 태그 선택, AND/OR 모드 선택

### 5. 태그 통계
- "📊 태그 통계" 버튼 클릭
- 전체 태그 사용 현황 확인

### 6. 기타 분류 기준
- 문서 개수가 적은 태그를 "기타" 폴더로 그룹화
- 기본값: 2건 미만은 기타로 분류
- 좌측 패널에서 최소 기준 조정 가능
