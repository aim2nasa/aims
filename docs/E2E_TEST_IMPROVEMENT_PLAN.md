**실제 UI 동작 방식:**
```
AllCustomersView.tsx:
```

## 개선 계획

### Phase 1: 삭제 테스트 수정 (우선)

**수정 방향:**
```typescript
// 삭제 모드 토글 버튼 클릭 (개발자 모드에서만 표시)
const deleteToggle = page.locator('button[aria-label="삭제"]');
await deleteToggle.click();

// 체크박스로 고객 선택
const checkbox = page.locator(`.customer-item:has-text("${customerName}") .customer-checkbox input`);
await checkbox.check();

// 삭제 버튼 클릭
const deleteButton = page.locator('button:has-text("삭제"):not([aria-label="삭제"])');
await deleteButton.click();

// 확인 모달에서 삭제 클릭
const confirmDelete = page.locator('.modal button:has-text("삭제")');
await confirmDelete.click();
```
