# 브랜드 이름 아이디어

> 보험 설계사를 위한 지능형 문서 관리 시스템 브랜드명 검토

## 검토 기준
- 마법의 조력자 / 지혜로운 도우미 느낌
- 나만의 정보 보관소
- 너무 흔하지 않으면서 기억하기 쉬움
- 2-4음절, 발음 용이

## 현재 유력 후보

### 1위: 모아 (MOA) ⭐

```
MOA = My Own Archive
    = 나만의 문서 보관소
```

| 항목 | 평가 |
|------|------|
| 한글 의미 | "모으다" - 정보를 모아주는 서비스 |
| 영문 약어 | **M**y **O**wn **A**rchive |
| 발음 | 2음절, 한/영 모두 자연스러움 |
| 스펠링 | 3글자, 타이핑 편함 |
| 도메인 | moa.app, getmoa.io 등 조합 가능 |
| B2B 적합성 | 깔끔하고 전문적 |
