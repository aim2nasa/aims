## 1. AIMS의 목표

**AIMS (Agent Intelligent Management System)** 는 보험 설계사를 위한 **지능형 문서 관리 시스템**입니다.

### 핵심 목표

| 목표 | 설명 |
|------|------|
| **문서 자동화** | PDF 업로드 → OCR → AI 태깅/요약 → 벡터 검색 |
| **고객 통합 관리** | 고객 정보, 계약, 관계도, 지역별 조회 |
| **연차보고서 파싱** | Annual Report PDF → 계약 정보 자동 추출 |
| **AI 검색 (RAG)** | 하이브리드 검색 (키워드 + 의미 기반) |
| **개인 파일 관리** | Google Drive 스타일 폴더 시스템 |

### 도메인 모델

```
설계사 ─(1:N)─► 고객 ─(1:N)─► 문서
                  └─(0:N)─► 계약 ─(N:1)─► 보험상품 ─(N:1)─► 보험사
```

## 2. 현재 완성도 분석

### 종합 평가 (2025-12 기준)

| 영역 | 완성도 | 상태 |
|------|--------|------|
| **프론트엔드 품질** | 90% | ✅ 우수 (A-) |
| **기능 완성도** | 75% | 🟡 양호 |
| **테스트 커버리지** | 85% | ✅ 우수 (3,800+ 테스트) |
| **프로덕션 준비도** | 60% | 🟡 보통 |
| **운영 인프라** | 30% | 🔴 부족 |

### 완성된 기능 (✅)

| 기능 | 설명 |
|------|------|
| 고객 관리 | CRUD + Soft Delete + 복원 + 관계도 |
| 문서 처리 | 업로드 → OCR → 벡터 임베딩 자동화 |
| AI 검색 | 하이브리드 검색 (Entity + Vector + Reranker) |
| 연차보고서 | 큐 기반 파싱 + 계약 정보 추출 |
| 지역별 조회 | 네이버 지도 연동 + 클러스터링 |
| 저장 용량 | 티어별 쿼터 시스템 (5GB~무제한) |
| 개인 파일 | 폴더 구조 + 드래그앤드롭 |
| 계약 관리 | CRUD + 엑셀 일괄등록 |

### 미완성/블로커 (🔴)

| 항목 | 상태 | 설명 |
|------|------|------|
| **문서 일괄등록 Backend** | 0% | 프론트엔드 100% 완료, API 6개 미구현 |
| **JWT 인증 완성** | 부분 | 카카오 OAuth 연동됨, x-user-id 헤더 병행 사용 |
| **운영 인프라** | 전무 | Sentry, 백업, CI/CD, Admin 대시보드 없음 |
| **Rate Limit 대응** | 부분 | AR 파싱 시 OpenAI 429 오류 빈번 (30초 대기 부족) |

### 미구현 Backend API (문서 일괄등록)

| 우선순위 | API Endpoint | 용도 |
|---------|--------------|------|
| 1 | `POST /api/customers/batch-lookup` | 고객명 일괄 조회 |
| 2 | `GET /api/users/me/storage` | 저장 용량 확인 |
| 3 | `POST /api/documents/batch-upload/start` | 배치 업로드 시작 |
| 4 | `POST /api/documents/batch-upload/:batchId/file` | 개별 파일 업로드 |
| 5 | `POST /api/documents/batch-upload/:batchId/complete` | 배치 완료 |
| 6 | `GET /api/documents/batch-upload/history` | 업로드 이력 |

## 9. 결론

AIMS는 **프론트엔드 품질과 AI 기능에서 우수**하나, **운영 인프라 구축과 일부 Backend API 완성**이 프로덕션 배포의 선결 과제입니다.

설계사 입장에서 가장 시급한 기능은:
1. **문서 일괄 업로드** (매월 대량 문서 처리)
2. **모바일 접근성** (외근 업무 특성)
3. **알림 시스템** (계약 관리 효율화)
