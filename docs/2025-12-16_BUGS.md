# AR 파싱 관련 이슈 목록

## 미해결 버그

### 1. 문서 탭 실시간 새로고침 안 됨
- **증상**: AR 문서 업로드 후 문서 탭에서 즉시 안 보임, F5 필요
- **예시**: 정부균 AR 문서가 문서 탭에 안 보임 → F5 후 보임
- **상태**: ❌ 미해결

### 2. AR 상태 동기화 버그
- **증상**: 동일 AR 문서가 테이블에서 "실패"로 표시되면서 동시에 "분석 중" 섹션에 "대기 중"으로도 표시됨
- **예시**: 안영미 문서가 두 곳에 동시 표시
- **상태**: ❌ 미해결

### 3. 재시도 후 Annual Report 탭 자동 새로고침 안 됨
- **증상**: 재시도 버튼 클릭 후 파싱 완료되어도 F5 필요
- **상태**: ❌ 미해결 (폴링 로직 추가했으나 검증 필요)

### 4. 재시도 버튼 호버 없이 안 보임
- **증상**: "실패" 배지 옆 재시도 버튼이 호버해야만 보임
- **상태**: ⚠️ 수정했으나 검증 필요 (컬럼 너비 90px→110px)

### 5. Rate Limit 오류
- **증상**: 여러 AR 동시 파싱 시 Rate limit 오류
- **상태**: ⚠️ 수정했으나 **서버 배포 필요** (30초 대기 추가, main.py)

## 해결된 이슈

### ✅ AR 삭제 시 파싱 결과 자동 삭제
- **증상**: AR 문서 삭제해도 파싱 결과가 남아있음
- **해결**: customer_name + issue_date 매칭으로 수정
