# 2025년도 예비창업패키지 사업계획서

> **[별첨 1] 양식 기준 작성 — 실제 제출 시 공식 HWP/DOCX 양식에 옮겨 작성할 것**

---

## 창업아이템 개요

| 항목 | 내용 |
|------|------|
| **제품·서비스명** | AIMS (Agent Intelligent Management System) |
| **분야** | 정보·통신 (AI 기반 SaaS) |
| **아이템 유형** | AI 지능형 문서 관리 플랫폼 |
| **신청 금액** | 50,000,000원 |
| **핵심 기술** | RAG(검색증강생성), 벡터 검색, OCR, MCP(Model Context Protocol) |

### 한줄 요약

보험 설계사가 고객 문서(보험증권, Annual Report 등)를 업로드하면, AI가 자동으로 텍스트 추출·요약·분류하고 자연어 질의로 필요한 정보를 즉시 찾아주는 **AI 기반 지능형 문서 관리 SaaS 플랫폼**

## 1-2. 창업아이템 목표시장 현황 분석

### TAM-SAM-SOM 분석

```
TAM (전체 시장): 글로벌 IDP(지능형 문서 처리) 시장
  └─ 2025년 $3.0B → 2032년 $66.7B (CAGR 33.4%, Research Nester)

SAM (유효 시장): 국내 보험 문서 관리 시장
  └─ 보험 설계사 39만 명 × 연 60만원(월 5만원) = 약 2,340억원

SOM (목표 시장): 초기 3년 내 획득 가능 시장
  └─ GA 소속 설계사 중 디지털 수용층 5,000명 × 월 49,000원
  └─ 연 약 29.4억원
```

### 직접 경쟁사 분석

| 구분 | Reducto | Unstructured.io | 보험사 내부 시스템 | **AIMS** |
|------|---------|-----------------|----------------|----------|
| **타겟** | 대기업 개발팀 | 개발자 | 전속 설계사 | **개인 설계사/GA** |
| **가격** | $300+/월 | 엔터프라이즈 | 무료(전속 한정) | **$49/월** |
| **문서 처리** | API만 제공 | API만 제공 | 자사 문서만 | **전사 통합** |
| **AI 검색** | 없음 | 기본 | 없음 | **RAG 기반** |
| **고객 관리** | 없음 | 없음 | 기본 | **통합 CRM** |
| **보험 특화** | 없음 | 없음 | 자사만 | **AR/CRS 자동 파싱** |
| **사용 난이도** | 개발자 필요 | 개발자 필요 | 학습 필요 | **비개발자 친화** |

### 경쟁 우위 요약

**1. 가격**: 경쟁사 대비 1/6 ~ 1/1000 수준
**2. 접근성**: 개발자 없이도 가입 후 즉시 사용
**3. 특화**: 보험 설계사 업무 프로세스에 최적화된 유일한 솔루션
**4. 통합**: 문서 관리 + 고객 관리 + AI 검색이 하나의 플랫폼

# 2. 실현가능성

## 2-1. 창업아이템 현황 (준비 정도)

### 개발 완료 현황 (MVP 이상)

AIMS는 이미 **프로덕션 운영 중**인 제품으로, 핵심 기능의 대부분이 개발·검증 완료되었습니다.

| 모듈 | 상태 | 완성도 | 설명 |
|------|------|--------|------|
| **프론트엔드** | 운영 중 | 95% | React 19 + TypeScript, 617개 컴포넌트 |
| **백엔드 API** | 운영 중 | 90% | Node.js + Express, 53,000+ 라인 |
| **문서 파이프라인** | 운영 중 | 90% | FastAPI, 업로드→OCR→요약→임베딩 자동화 |
| **AI 어시스턴트** | 운영 중 | 80% | MCP 기반 46개 도구, 자연어 질의 |
| **RAG 검색** | 운영 중 | 85% | 하이브리드(벡터+키워드) 검색, GPT 재순위화 |
| **AR/CRS 파서** | 운영 중 | 75% | pdfplumber 기반 테이블 자동 추출 |
| **AutoClicker** | 운영 중 | 90% | PDF 자동 다운로드 도구 |
| **인증 시스템** | 운영 중 | 95% | JWT + 카카오/네이버/구글 소셜 로그인 |

AIMS의 핵심 기능은 이미 개발 완료되어 있으므로, 협약 기간에는 **상용화 고도화**에 집중합니다.

| 작업 | 상세 | 산출물 |
|------|------|--------|
| SaaS 인프라 구축 | AWS/NCP 멀티테넌시 배포 | 클라우드 인프라 |

## 4-2. 중장기 사회적 가치 도입계획

### 사회적 가치 창출 방향

**1. 보험 설계사 디지털 역량 격차 해소**

- 문제: 고령 설계사(50대 이상)의 디지털 도구 활용 어려움
- 해결: AIMS의 직관적 UI + AI 어시스턴트로 기술 장벽 제거
- 효과: 연령에 관계없이 동등한 업무 효율 달성

**2. 보험 소비자 보호 기여**

- 문제: 설계사가 고객의 전체 보험 현황을 파악하지 못해 불완전판매 발생
- 해결: AIMS의 통합 문서 관리로 고객의 보험 현황 정확히 파악
- 효과: 불완전판매 감소, 적정 보험 설계 → 소비자 이익 증대

**3. 1인 사업자 경쟁력 강화**

- 문제: 대형 GA만 IT 시스템 투자 가능, 소규모 설계사는 수작업 의존
- 해결: 월 49,000원의 저렴한 비용으로 기업급 AI 문서 관리 제공
- 효과: IT 투자 여력이 없는 소규모 설계사도 대형 GA 수준의 업무 환경 확보

**4. 일자리 창출**
