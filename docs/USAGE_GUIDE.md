# AIMS 사용 가이드

> 최종 업데이트: 2026-04-02 | 리뷰: Alex(기술), Gini(품질), Sora(설계사 관점)

---

## 1. 시작하기

### 이 서비스는 무엇인가요?

보험 설계사를 위한 AI 기반 지능형 문서 관리 시스템입니다.

- 고객 정보와 보험 계약을 체계적으로 관리합니다
- AR/CRS 문서를 AI가 자동으로 분석하여 계약 정보를 추출합니다
- 업로드된 문서를 AI가 자동으로 분류하고, 알아보기 쉬운 별칭을 만들어줍니다
- AI 어시스턴트에게 자연어로 질문하고 업무를 요청할 수 있습니다
- PC, 모바일, 태블릿 어디서든 접속하여 사용 가능합니다

### 첫 번째 고객 등록하기

처음 사용하신다면 이 가이드를 따라 첫 고객을 등록해보세요.

1. 왼쪽 메뉴에서 "빠른 작업"을 클릭합니다
2. AR(Annual Report)이 있다면 "고객·계약·문서 등록"을 선택합니다
3. AR이 없다면 "고객 수동등록"을 선택합니다
4. 고객 정보를 입력하고 저장을 클릭합니다

---

## 2. 고객·계약·문서 등록

### AR 문서로 고객과 계약 자동 등록

연례보고서(Annual Report)를 업로드하면 AI가 자동으로 피보험자 정보와 계약 내역을 추출합니다.

1. "고객·계약·문서 등록" 화면에서 AR 파일을 드래그앤드롭합니다
2. AI가 피보험자 이름과 발행일을 자동으로 추출합니다
3. 같은 이름의 기존 고객이 있으면 선택하거나 새 고객으로 등록합니다
4. 파싱이 완료되면 계약 정보가 보험계약 탭에 자동으로 표시됩니다

### 증권/청약서 문서 등록하기

1. 먼저 상단에서 문서를 연결할 고객을 선택합니다
2. 파일 업로드 영역에 문서를 드래그앤드롭합니다
3. PDF, 이미지 등 다양한 형식을 지원합니다
4. 업로드된 문서는 해당 고객의 문서 탭에서 확인할 수 있습니다

### AR 파싱 결과 확인하기

1. 고객 상세 화면에서 "Annual Report" 탭을 선택합니다
2. AR 목록에서 확인할 항목을 클릭합니다
3. "상세 보기"로 추출된 계약 정보를 확인합니다
4. 필요시 우클릭 메뉴에서 "보험계약 등록"을 선택하여 수동 등록할 수 있습니다

## 3. 고객 수동등록

### 개인 고객 등록하기

1. "고객 수동등록" 화면에서 "개인" 탭을 선택합니다
2. 고객명(필수), 휴대폰, 이메일, 주소 등을 입력합니다
3. 생년월일과 성별을 입력하면 보험 상담 시 유용합니다
4. "등록" 버튼을 클릭하여 저장합니다

### 법인 고객 등록하기

1. "고객 수동등록" 화면에서 "법인" 탭을 선택합니다
2. 법인명(필수), 대표전화, 이메일을 입력합니다
3. 사업자번호와 대표자명을 입력합니다
4. 사업장 주소를 입력하고 "등록" 버튼을 클릭합니다

---

## 4. 고객 일괄등록

### 엑셀 파일 준비하기

1. 엑셀 파일(.xlsx)에 최대 3개의 시트를 만들 수 있습니다: 개인고객, 법인고객, 계약
2. 개인고객 시트: 고객명(필수), 이메일, 휴대폰, 주소, 성별, 생년월일
3. 법인고객 시트: 고객명(필수), 이메일, 대표전화, 주소, 사업자번호, 대표자명
4. 날짜는 YYYY-MM-DD 형식(예: 2024-03-15)을 권장합니다

### 엑셀 파일 업로드하기

1. "고객 일괄등록" 화면에서 엑셀 파일을 드래그앤드롭합니다
2. 시스템이 자동으로 시트별 데이터를 읽어옵니다
3. 고객명 빈값, 동명이인 등 오류가 있으면 빨간색으로 표시됩니다
4. 상품명 매칭 상태: 녹색(정확), 노란색(정규화), 빨간색(미매칭)

### 검증 후 등록하기

1. 모든 오류를 수정한 후 "등록" 버튼을 클릭합니다
2. 개인고객, 법인고객, 계약 순서로 등록이 진행됩니다
3. 등록 완료 후 결과 요약이 표시됩니다
4. "전체 고객 보기"에서 등록된 고객을 확인할 수 있습니다

## 5. 문서 일괄등록

### 폴더 구조 준비하기

1. 루트 폴더 아래에 고객명과 동일한 이름의 폴더를 만듭니다
2. 각 고객 폴더 안에 해당 고객의 문서 파일들을 넣습니다
3. 예: /루트/홍길동/증권.pdf, /루트/홍길동/청약서.pdf
4. 폴더명이 시스템에 등록된 고객명과 일치해야 자동 매핑됩니다

### 폴더 업로드하기

1. "문서 일괄등록" 화면에서 루트 폴더를 드래그앤드롭합니다
2. 시스템이 폴더명과 기존 고객명을 자동 매핑합니다
3. 매핑 결과를 검토하고, 필요시 수동으로 고객을 선택합니다
4. "업로드 시작" 버튼을 클릭하여 일괄 업로드합니다

---

## 6. 전체 고객 보기

### 고객 목록 조회하기

1. 왼쪽 메뉴에서 "고객" > "전체 고객 보기"를 클릭합니다
2. 검색창에 이름, 전화번호, 이메일로 검색할 수 있습니다
3. 초성 필터(ㄱ, ㄴ, ㄷ...)로 빠르게 찾을 수 있습니다
4. 상태 필터(활성, 휴면)로 고객을 분류하여 볼 수 있습니다
5. 지역별, 관계별 보기로 전환하여 다양한 관점으로 고객을 볼 수 있습니다

### 고객 상세 정보 확인하기

1. 고객 목록에서 원하는 고객을 클릭합니다
2. 오른쪽 패널에서 기본 정보를 확인할 수 있습니다
3. 더블클릭하면 전체 화면에서 문서, 계약, AR 정보를 모두 볼 수 있습니다
4. 우클릭 메뉴로 전화, 문자, 휴면 처리 등을 할 수 있습니다

### 우클릭 메뉴 활용하기

1. 고객 목록에서 원하는 고객을 우클릭(마우스 오른쪽 버튼)합니다
2. "전화하기": 등록된 전화번호로 바로 연결합니다
3. "문자 보내기": 등록된 번호로 문자 메시지를 보냅니다
4. "휴면 처리": 활성 고객을 휴면 상태로 변경합니다
5. "상세 보기": 고객 전체 화면 상세 정보를 엽니다

## 6.5. 고객 보기 방식

### 지역별 고객 보기

1. 왼쪽 메뉴에서 "고객" > "지역별 고객 보기"를 클릭합니다
2. 시/도 > 시/군/구 단위로 고객이 자동 분류됩니다
3. 지역을 클릭하면 해당 지역의 고객 목록이 표시됩니다
4. 지역별 고객 수를 한눈에 파악할 수 있습니다

### 관계별 고객 보기

1. 왼쪽 메뉴에서 "고객" > "관계별 고객 보기"를 클릭합니다
2. 가족 관계(배우자, 자녀 등)나 법인 관계로 묶인 고객 그룹이 표시됩니다
3. 그룹을 클릭하면 구성원과 관계를 확인할 수 있습니다
4. 관계가 설정되지 않은 고객은 별도로 표시됩니다

## 7. 고객별 문서함

### 고객별 문서함 사용하기

고객별로 분류된 문서를 폴더 형태로 탐색합니다.

1. 왼쪽 메뉴에서 "문서" > "고객별 문서함"을 클릭합니다
2. 고객명을 클릭하면 해당 고객의 문서가 유형별로 정리되어 표시됩니다
3. 고객명 옆 ··· 버튼으로 문서함 다운로드, 상세 보기 등을 할 수 있습니다
4. 초성 필터(ㄱ, ㄴ, ㄷ...)로 특정 고객을 빠르게 찾을 수 있습니다

### 검색과 필터

1. 검색창에 파일명 또는 고객명으로 검색합니다
2. "오늘" 버튼으로 당일 업로드 문서만 필터링합니다
3. 날짜 이동 기능으로 특정 날짜의 문서로 바로 이동합니다
4. 헤더의 날짜/크기 등을 클릭하여 정렬합니다

## 7.5. 상세 문서검색

### 문서 내용 검색하기

파일명뿐 아니라 문서 안의 내용까지 검색합니다.

1. 왼쪽 메뉴에서 "문서" > "상세 문서검색"을 클릭합니다
2. 검색창에 찾고 싶은 내용을 입력합니다
3. 문서 제목뿐 아니라 문서 본문 내용까지 검색됩니다
4. 검색 결과에서 문서를 클릭하면 미리보기가 표시됩니다

### AI 검색 활용하기

AI가 질문의 의미를 이해하여 관련 문서를 찾아줍니다.

1. 검색창에 자연어로 질문합니다
2. AI가 질문의 의미를 분석하여 관련도가 높은 문서를 우선 표시합니다
3. 정확한 단어가 아니어도 비슷한 내용의 문서를 찾을 수 있습니다
4. 검색 결과가 많으면 고객명, 문서유형 등으로 추가 필터링이 가능합니다

## 7.7. 별칭AI

### 별칭AI란?

AI가 문서 내용을 분석하여 알아보기 쉬운 이름을 자동으로 만들어줍니다.

1. 업로드된 문서의 원본 파일명은 길거나 의미를 알기 어려운 경우가 많습니다
2. 별칭AI는 문서 내용을 분석하여 의미 있는 이름을 생성합니다
3. 별칭과 원본 파일명 중 어느 것으로 표시할지 언제든 전환할 수 있으며 원본은 보존됩니다
4. 전체 문서 보기, 고객별 문서함 등에서 "별칭AI" 버튼으로 생성할 수 있습니다

### 별칭 생성하기

1. 전체 문서 보기에서 "별칭AI" 버튼을 클릭하면 별칭 생성 모드에 진입합니다
2. 별칭을 생성할 문서를 체크박스로 선택합니다
3. 문서를 선택한 후 버튼이 "완료"로 바뀐 상태에서 클릭하면 AI가 별칭을 생성합니다
4. 선택 없이 "완료"를 클릭하면 별칭 생성 없이 모드만 종료됩니다
5. 파일명 옆 전환 버튼으로 별칭/원본을 바꿔볼 수 있습니다

## 8. 전체 문서 보기

### 문서 목록 조회하기

1. 왼쪽 메뉴에서 "문서" > "전체 문서 보기"를 클릭합니다
2. 검색창에 파일명, 고객명으로 검색할 수 있습니다
3. 초성 필터로 고객명 기준 빠른 검색이 가능합니다
4. 상단 탭(전체/처리중/완료/에러)으로 문서 처리 상태별로 필터링할 수 있습니다
5. 별칭AI 버튼으로 선택한 문서에 AI가 알아보기 쉬운 이름을 자동으로 만들어줍니다

### 문서 미리보기

1. 문서 목록에서 원하는 문서를 클릭합니다
2. 오른쪽 패널에서 문서 미리보기가 표시됩니다
3. 더블클릭하면 모달 창에서 크게 볼 수 있습니다
4. 우클릭 메뉴로 다운로드, 고객 연결, 삭제 등을 할 수 있습니다

### 파일명 별칭/원본 전환

1. 문서 목록 상단의 파일명 전환 버튼을 클릭합니다
2. "별칭" 모드: AI가 만들어준 별칭이 표시됩니다
3. "원본" 모드: 업로드 시 원본 파일명이 표시됩니다
4. 전체 문서 보기, 상세 문서 검색, 고객별 문서함 모두에서 동일하게 사용 가능합니다
5. 검색 시에도 선택한 모드에 맞춰 파일명 기준으로 검색됩니다

### 문서 처리 상태 이해하기

1. 문서를 업로드하면 자동으로 처리가 시작됩니다
2. 1단계: 텍스트 추출 — 문서에서 글자를 읽어냅니다 (스캔 문서는 자동으로 글자 인식 처리)
3. 2단계: AI 분류 — 문서 유형(보험증권, 청약서, 신분증 등)을 자동 판별합니다
4. 3단계: AI 요약 — 문서 내용을 요약하고 검색용 데이터를 생성합니다
5. 모든 처리가 완료되면 상태가 "완료"로 표시됩니다
6. 처리 중 오류가 발생하면 "에러" 탭에서 확인할 수 있으며, 대부분 자동으로 재시도됩니다

## 9. 용어 설명

### AR (Annual Report)
- AR은 보험사가 매년 피보험자에게 발송하는 연례보고서입니다
- 가입된 모든 보험 계약의 요약 정보가 포함되어 있습니다
- AR을 업로드하면 AI가 자동으로 계약 정보를 추출합니다
- 같은 고객의 여러 연도 AR을 업로드하면 계약 변화를 추적할 수 있습니다

### 피보험자와 계약자
- 피보험자: 보험 사고 발생 시 보험금을 받는 대상
- 계약자: 보험 계약을 체결하고 보험료를 납입하는 사람
- 피보험자와 계약자는 동일인일 수도, 다른 사람일 수도 있습니다
- 이 서비스에서 "고객"은 주로 계약자를 의미합니다

### 활성 고객과 휴면 고객
- 활성 고객: 현재 관리 중인 고객으로 기본 목록에 표시됩니다
- 휴면 고객: 연락이 뜸하거나 더 이상 관리하지 않는 고객입니다
- 휴면 처리된 고객도 삭제되지 않고 "휴면" 필터에서 확인 가능합니다
- 휴면 고객은 언제든 다시 활성화할 수 있습니다

### CRS (Customer Review Service)
- CRS는 Customer Review Service의 약자로, 변액보험 리포트를 의미합니다
- 변액보험 계약자에게 정기적으로 발송되는 보고서입니다
- 계약사항, 보험료 납입현황, 펀드별 적립금/수익률 등이 포함됩니다
- AIMS에 업로드하면 AI가 자동으로 분석하여 "변액리포트" 탭에서 확인 가능합니다

## 10. AutoClicker

### AutoClicker란?
MetLife 고객의 CRS, AR PDF 파일들을 자동으로 다운로드하는 PC 프로그램입니다.

1. MetLife 홈페이지에서 고객별 PDF 문서를 자동으로 다운로드합니다
2. 다운로드된 PDF는 AIMS에 자동으로 업로드되어 분석됩니다
3. Windows PC에서만 사용 가능하며, 1920×1080 해상도를 권장합니다

### 설치 및 실행하기
1. AIMS 왼쪽 메뉴에서 "AutoClicker" 메뉴를 클릭합니다
2. "AutoClicker 실행" 버튼을 클릭합니다
3. 처음 사용 시 설치 프로그램이 자동으로 다운로드됩니다
4. 설치 완료 후 다시 "AutoClicker 실행" 버튼을 클릭하면 바로 실행됩니다

## 10.5. Annual Report (AR)

### AR 문서 업로드하기
1. "고객·계약·문서 등록" 화면에서 AR 파일을 드래그앤드롭합니다
2. AI가 자동으로 AR 문서를 감지하고 분석을 시작합니다
3. 문서 내의 모든 보험계약 정보가 자동으로 추출됩니다
4. 추출된 계약은 해당 고객의 "계약" 탭에서 확인할 수 있습니다

## 11. 변액리포트 (CRS)

### CRS 문서 업로드하기
1. "고객·계약·문서 등록" 화면에서 CRS 파일을 드래그앤드롭합니다
2. AI가 자동으로 CRS 문서를 감지하고 분석을 시작합니다
3. 분석이 완료되면 계약사항, 보험료, 펀드 구성 정보가 자동으로 추출됩니다

### 변액리포트 탭 사용법
1. 고객 상세 화면에서 "변액리포트" 탭을 클릭합니다
2. 계약자, 피보험자, 발행일, 상품명, 펀드 수, 상태 등 확인 가능
3. 완료된 항목을 클릭하면 상세 모달이 열립니다

## 13. AI 어시스턴트

### 사용법
1. "[고객명] 고객 등록해줘" — 고객 등록을 요청합니다
2. "[고객명]의 계약 정보 알려줘" — 특정 고객의 계약을 조회합니다
3. "[고객명] 고객의 문서 중 보험증권 찾아줘" — 문서를 검색합니다
4. "[고객명] 고객에게 메모 남겨줘: 다음주 상담 예정" — 메모를 추가합니다
5. 대화 내역은 저장되어 페이지를 새로고침해도 유지됩니다
6. 별도 창(팝업) 아이콘으로 AI 어시스턴트를 넓은 화면에서 사용할 수 있습니다

## 14. 모바일 사용법

### 모바일 접속 안내
1. 모바일 기기의 웹 브라우저에서 AIMS 주소로 접속합니다
2. 화면이 자동으로 모바일에 최적화된 레이아웃으로 전환됩니다
3. 가로 모드와 세로 모드 모두 지원합니다

### 모바일 주요 기능
1. 고객 목록 조회 및 상세 정보 확인이 가능합니다
2. 문서 미리보기와 조회가 가능합니다
3. AI 어시스턴트를 전체 화면으로 사용할 수 있습니다
4. 초성 필터, 검색 등 PC와 동일한 필터링 기능을 사용할 수 있습니다
