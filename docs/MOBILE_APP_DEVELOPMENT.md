# AIMS 모바일 앱 개발

## 개요

AIMS 시스템의 모바일 앱 개발. 웹 AI 어시스턴트와 동일한 기능을 채팅/음성으로 제공.

**시작일**: 2025.12.22

---

## 결정 사항

| 항목 | 선택 |
|------|------|
| 기술 스택 | React Native + Expo |
| 음성 기능 | STT + TTS 모두 |
| 배포 방식 | 내부 배포 (TestFlight/내부 테스트) |

---

## UI/UX 디자인

### 디자인 컨셉

**웹 AI 어시스턴트 + ChatGPT 결합**

| 요소 | 웹 AI 어시스턴트 | ChatGPT | 모바일 앱 |
|------|-----------------|---------|----------|
| 테마 | 다크 모드 | 라이트/다크 | 다크 모드 (기본) |
| 시작 화면 | 기능 카드 목록 (17개) | 빈 채팅창 | **기능 카드 + 빈 채팅** |
| 입력 방식 | 텍스트만 | 텍스트 + 음성 | **텍스트 + 음성** |
| 대화 히스토리 | 드롭다운 | 사이드바 | **바텀시트** |

### MCP Tools (18개)

| 카테고리 | Tool | 설명 |
|----------|------|------|
| **고객 관리** | `search_customers` | 고객 검색 |
| | `get_customer` | 고객 상세 조회 |
| | `create_customer` | 고객 등록 |
| | `update_customer` | 고객 정보 수정 |
| **계약 관리** | `list_contracts` | 계약 목록 조회 |
| | `get_contract_details` | 계약 상세 조회 |
| **일정** | `find_birthday_customers` | 생일 고객 |
| | `find_expiring_contracts` | 만기 예정 계약 |
| **문서** | `search_documents` | 문서 AI 검색 |
| | `get_document` | 문서 상세 |
| | `list_customer_documents` | 고객별 문서 조회 |
| **메모** | `add_customer_memo` | 메모 추가 |
| | `list_customer_memos` | 메모 조회 |
| | `delete_customer_memo` | 메모 삭제 |
| **통계** | `get_statistics` | 전체 통계/월별 현황 |
| **관계** | `get_customer_network` | 고객 관계 조회 |
| **상품** | `search_products` | 보험상품 검색 |
| | `get_product_details` | 상품 상세 정보 |

- [x] 인증 스토어 구현 (`src/stores/authStore.ts`)
- [x] 기본 화면 UI 구현 (로그인, 채팅, 음성, 설정)

**생성된 파일:**
```
mobile/aims-mobile/
├── app/
│   ├── _layout.tsx          # 루트 레이아웃
│   ├── index.tsx            # 로그인 화면
│   └── (auth)/
│       ├── _layout.tsx      # 탭 네비게이션
│       ├── chat/index.tsx   # 채팅 화면 (18개 MCP Tools 카테고리)
│       ├── voice/index.tsx  # 음성 어시스턴트
│       └── settings/index.tsx
├── src/
│   ├── types/index.ts       # TypeScript 타입
│   ├── services/api.ts      # API 클라이언트 (SSE 스트리밍)
│   ├── stores/authStore.ts  # Zustand + SecureStore
│   └── utils/theme.ts       # 색상, 간격, 폰트 테마
└── app.json                 # Expo 설정 (마이크 권한 등)
```
