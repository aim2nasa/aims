# AIMS 페이지 이름 정의

## 고객

| 이름 | URL/위치 | 메뉴 경로 | 설명 |
|------|----------|----------|------|
| 고객전체페이지 | `?view=customers-all` (메인 영역) | 고객 > 전체 고객 보기 | 모든 고객의 종합 정보를 한눈에 보여주는 페이지 (이름, 생년월일, 성별, 휴대폰, 이메일, 주소, 상태, 등록일) |
| 고객상세페이지 | `?view=customers-full-detail` (메인 영역) | 고객전체페이지에서 고객 클릭 | 특정 고객 1명의 기본정보, 관계자, 문서, 계약을 모두 보여주는 상세 페이지 |
| 고객미니페이지 | RightPane | 고객 클릭 시 RP에 표시 | 고객상세페이지의 축소 버전을 RightPane에 표시 |
| 고객문서분류함 | `?view=customer-document-explorer` (메인 영역) | 고객상세페이지 문서탭 → 문서 분류함 버튼 | 특정 고객의 문서를 대분류→소분류 트리 구조로 탐색하는 페이지 (내 문서/관계자 문서 탭) |

## 문서

| 이름 | URL/위치 | 메뉴 경로 | 설명 |
|------|----------|----------|------|
| 전체문서보기 | `?view=documents-library` (메인 영역) | 문서 > 전체 문서 보기 | 모든 문서를 목록으로 보여주는 페이지 |
| 고객별문서함 | `?view=documents-explorer` (메인 영역) | 문서 > 고객별 문서함 | 고객별 폴더를 나열하고, 고객 폴더를 열면 해당 고객의 문서 목록이 표시되는 페이지 |
| 상세문서검색 | `?view=documents-search` (메인 영역) | 문서 > 상세 문서검색 | 상세 조건으로 문서를 검색하는 페이지 |

### 고객상세페이지 섹션 구조

```
┌─────────────────────────────┬─────────────────────────────┐
│  고객정보 섹션 (좌상)         │  계약 섹션 (우상)             │
│  [고객 정보] [메모]           │  [AR 이력] [변액 리포트 이력]  │
│  - 기본정보 (이름,생년,성별…)  │  [법인 계약]                  │
│  - 관계자 목록                │  - 계약 목록 / 상품 검색       │
├─────────────────────────────┤                               │
│  문서 섹션 (좌하)             ├─────────────────────────────┤
│  - 문서 목록 + 검색           │  리포트 섹션 (우하)           │
│  - 분류 필터, 정렬            │  [Annual Report] [변액 리포트] │
│  - 문서 분류함 버튼           │  - AR/변액 리포트 뷰어        │
└─────────────────────────────┴─────────────────────────────┘
```

| 섹션 이름 | 위치 | 포함 탭 | 설명 |
|----------|------|--------|------|
| 고객정보 섹션 | 좌상 | 고객 정보, 메모 | 기본정보 + 관계자 목록 + 메모 |
| 문서 섹션 | 좌하 | (탭 없음) | 문서 목록, 검색, 분류 필터, 문서 분류함 진입 |
| 계약 섹션 | 우상 | AR 이력, 변액 리포트 이력, 법인 계약 | 계약/이력 목록 |
| 리포트 섹션 | 우하 | Annual Report, 변액 리포트 | AR/변액 리포트 뷰어 |

### UI 요소 명칭

| 명칭 | 설명 | 예시 |
|------|------|------|
| 섹션 | 페이지 내 독립된 영역 | 고객정보 섹션, 문서 섹션 |
| 탭 바 | 섹션 상단의 탭 전환 버튼 + 부가 도구(검색, 필터 등)가 있는 바 | 고객정보 섹션의 탭 바 ([고객 정보] [메모]) |
| 탭 | 탭 바 안의 개별 전환 버튼 | 고객 정보 탭, 메모 탭, AR 이력 탭 |
| 탭 콘텐츠 | 탭 선택 시 표시되는 콘텐츠 영역 | 고객 정보 탭의 기본정보 + 관계자 목록 |
| 테이블 헤더 | 테이블 콘텐츠의 컬럼명 행 (sticky) | 관계 유형 / 관련 고객 / 등록일 / 관계 삭제 |

#### 지칭 예시

- "고객정보 섹션의 탭 바에 [고객 정보]와 [메모] 탭이 있다"
- "문서 섹션의 탭 바에 검색창과 분류 필터가 있다"
- "계약 섹션의 [AR 이력] 탭 콘텐츠에 계약 목록이 표시된다"
- "관계자 목록의 테이블 헤더가 스크롤 시 고정(sticky)된다"
