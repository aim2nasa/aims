# AIMS 자주 묻는 질문 (FAQ)

> 최종 업데이트: 2026-04-02 | 총 46개 항목

---

## 일반

1. **이 서비스는 무엇인가요?**
   보험 설계사를 위한 지능형 문서 관리 시스템입니다. 고객 정보, 보험 계약, 관련 문서를 체계적으로 관리하고, AI가 AR(연례보고서)을 자동으로 분석하여 계약 정보를 추출합니다.

2. **처음 사용하는데, 어디서 시작하면 되나요?**
   왼쪽 메뉴의 "빠른 작업"에서 시작하세요. AR 문서가 있다면 "고객·계약·문서 등록"을, 없다면 "고객 수동등록"을 선택하여 첫 고객을 등록할 수 있습니다.

3. **화면을 어두운 모드로 바꿀 수 있나요?**
   네, 상단 헤더 우측의 태양/달 아이콘 사이에 있는 전환 버튼을 클릭하면 라이트/다크 모드를 바꿀 수 있습니다. 설정은 자동으로 저장됩니다.

4. **파일 크기 제한이 있나요? 어떤 파일 형식을 지원하나요?**
   PDF, JPG, PNG 등 이미지, Word(DOC/DOCX), Excel(XLS/XLSX), PowerPoint(PPT/PPTX), HWP 등 대부분의 문서 형식을 지원합니다. 파일 크기는 건당 50MB까지 업로드 가능합니다.

5. **내가 올린 문서를 다른 설계사가 볼 수 있나요?**
   아닙니다. AIMS는 설계사별로 완전히 격리되어 있습니다. 다른 설계사가 내 고객이나 문서를 볼 수 없으며, 나도 다른 설계사의 데이터를 볼 수 없습니다.

## 고객·계약·문서 등록

1. **AR 문서를 업로드했는데 고객이 자동 생성되지 않아요.**
   AR에서 추출한 피보험자 이름과 유사한 기존 고객이 있을 경우, 시스템이 고객 선택 팝업을 표시합니다. 기존 고객을 선택하거나 "새 고객으로 등록"을 선택하세요.

2. **AR 파싱에 실패했어요. 어떻게 해야 하나요?**
   AR 문서가 스캔본이거나 이미지 품질이 낮으면 파싱에 실패할 수 있습니다. 고객 상세 > Annual Report 탭에서 해당 AR을 우클릭하고 "재시도"를 선택하세요.

3. **증권/청약서를 특정 고객에게 연결하려면 어떻게 하나요?**
   "고객·계약·문서 등록" 화면 상단에서 먼저 고객을 선택한 후 파일을 업로드하세요. 또는 "전체 문서 보기"에서 문서를 우클릭하고 "고객 연결"을 선택하여 나중에 연결할 수도 있습니다.

4. **AR에서 추출된 계약 정보가 보험계약 탭에 안 보여요.**
   AR 파싱이 완료되면 계약 정보가 자동으로 보험계약 탭에 등록됩니다. 파싱 상태가 "완료"인지 확인하세요.

5. **문서 상태가 '처리중'에서 변하지 않아요.**
   문서 처리는 보통 1~2분 내에 완료됩니다. 스캔 문서나 대용량 파일은 글자 인식 처리에 시간이 더 걸릴 수 있습니다. 10분 이상 지속되면 페이지를 새로고침해보세요.

6. **문서 상태가 '에러'로 표시되어요.**
   문서 처리 중 오류가 발생한 경우입니다. 대부분 자동으로 재시도됩니다. 계속 에러 상태가 유지되면 해당 문서를 삭제 후 다시 업로드하거나 1:1 문의를 이용해주세요.

7. **문서를 잘못 올렸는데 어떻게 삭제해요?**
   전체 문서 보기에서 휴지통 아이콘을 클릭하면 삭제 모드에 진입합니다. 삭제할 문서를 체크박스로 선택한 후 삭제 버튼을 클릭하세요.

8. **문서를 다른 고객에게 옮길 수 있나요?**
   네, 문서 목록에서 해당 문서를 우클릭하고 연결 고객 변경을 선택하면 다른 고객으로 변경할 수 있습니다.

9. **AI가 문서 종류를 잘못 분류했어요. 고칠 수 있나요?**
   문서 상세 화면에서 문서 유형을 수동으로 변경할 수 있습니다. 문서를 클릭하여 상세 정보를 열고, 문서 유형 영역을 클릭하여 올바른 유형을 선택하세요.

## 고객 등록

1. **같은 이름의 고객을 등록할 수 있나요?**
   같은 설계사 내에서 고객명은 중복될 수 없습니다. 동명이인이 있다면 "홍길동(일산)", "홍길동(분당)" 등으로 구분하여 등록하세요.

2. **법인 고객의 대표전화와 개인 휴대폰 번호 형식이 다른가요?**
   개인 고객은 010-XXXX-XXXX 형식, 법인 고객은 02-XXXX-XXXX 등 지역번호 포함 형식으로 입력합니다.

## 고객 조회

1. **휴면 고객은 어디서 볼 수 있나요?**
   "전체 고객 보기"에서 상단 필터 버튼 중 "휴면"을 클릭하면 휴면 처리된 고객만 표시됩니다.

2. **고객 목록에서 초성 검색은 어떻게 하나요?**
   검색창 아래 초성 필터 바에서 원하는 초성(ㄱ, ㄴ, ㄷ...)을 클릭하세요.

3. **가족계약 탭은 어디에 있나요?**
   고객 상세 화면에서 "가족계약" 탭을 클릭하면 가족 구성원 전체의 보험계약을 통합 조회할 수 있습니다. 가족관계가 설정된 고객만 활성화됩니다.

4. **법인계약 탭은 무엇인가요?**
   고객 상세 화면의 "법인계약" 탭에서는 법인 관련 계약을 자동으로 감지하여 보여줍니다.

## 문서 조회

1. **고객별 문서함과 전체 문서 보기의 차이점은 무엇인가요?**
   전체 문서 보기는 모든 문서를 날짜순으로 나열합니다. 고객별 문서함은 고객 이름 아래에 서류가 종류별로 정리되어 있습니다.

2. **고객별 문서함에서 썸네일 미리보기를 켜고 끄려면?**
   상단 툴바에서 썸네일 아이콘을 클릭하여 켜고 끌 수 있습니다.

3. **파일명을 별칭과 원본으로 전환할 수 있나요?**
   네. 파일명 옆의 전환 버튼을 클릭하면 별칭과 원본 파일명 사이를 전환할 수 있습니다.

4. **별칭이란 무엇인가요?**
   별칭은 AI가 문서 내용을 분석하여 자동으로 만들어주는 알아보기 쉬운 파일명입니다. 원본 파일명은 그대로 보존됩니다.

5. **별칭이 생성되지 않는 문서가 있어요.**
   ZIP, 일러스트레이터 파일 등 텍스트를 추출할 수 없는 파일은 원본 파일명을 기반으로 별칭을 생성합니다. 파일명이 너무 짧으면 생성할 수 없습니다.

6. **상세 문서검색과 일반 검색의 차이가 무엇인가요?**
   일반 검색은 파일명 기준, 상세 문서검색은 문서 본문 내용까지 검색합니다. AI가 질문의 의미를 이해하여 관련 문서를 찾아줍니다.

7. **문서가 안 보여요. 문서를 찾을 수 없어요.**
   상태 탭이 "전체"로 되어 있는지, 검색어가 입력되어 있지 않은지 확인하세요. 고객에게 연결하지 않은 문서는 고객별 문서함에서 표시되지 않습니다.

## 일괄 등록

1. **엑셀 일괄등록에서 상품명이 빨간색으로 표시되어요.**
   빨간색은 시스템에 등록되지 않은 상품명입니다. 정확한 상품명으로 수정 후 다시 업로드하세요.

2. **엑셀에서 날짜가 이상하게 표시되어요.**
   셀 서식을 "날짜"로 변경 후 저장하세요. YYYY-MM-DD 형식을 권장합니다.

3. **문서 일괄등록에서 폴더가 고객과 매핑되지 않아요.**
   폴더명이 시스템에 등록된 고객명과 정확히 일치해야 합니다.

## AutoClicker

1. **AutoClicker란 무엇인가요?**
   MetLife 고객의 CRS, AR PDF 파일들을 자동으로 다운로드하는 PC 프로그램입니다.

2. **AutoClicker 실행 버튼을 눌러도 반응이 없어요.**
   처음 사용 시 설치 프로그램이 다운로드됩니다. 설치 후 다시 실행 버튼을 클릭하세요.

3. **AutoClicker는 어떤 환경에서 사용할 수 있나요?**
   Windows PC에서만 사용 가능하며, 1920×1080 해상도를 권장합니다.

4. **AutoClicker는 자동으로 업데이트되나요?**
   네. 매 실행 시 최신 버전을 자동으로 확인하고 업데이트합니다.

## Annual Report

1. **Annual Report(AR)란 무엇인가요?**
   피보험자의 전체 보험계약을 요약한 연례보고서입니다.

2. **AR 문서를 업로드하면 어떻게 되나요?**
   AI가 자동으로 문서를 분석하여 모든 보험계약 정보를 추출, 등록합니다.

3. **CRS와 AR은 어떻게 다른가요?**
   AR은 전체 보험계약 요약, CRS는 변액보험 특화 상세 리포트입니다.

---

## 변액리포트 (CRS)

1. **변액리포트(CRS)란 무엇인가요?**
   변액보험 계약자에게 발송되는 정기 리포트로, 펀드별 적립금/수익률 등 상세 정보가 담겨 있습니다.

2. **CRS 문서를 업로드하면 어떻게 되나요?**
   AI가 자동으로 분석하여 "변액리포트" 탭에서 확인 가능합니다.

3. **CRS 파싱에 실패했어요.**
   재시도 버튼을 클릭하세요 (최대 3회). 계속 실패하면 1:1 문의를 이용해주세요.

## 모바일

1. **모바일에서도 AIMS를 사용할 수 있나요?**
   네. 모바일 브라우저에서 접속하면 자동으로 최적화된 화면으로 전환됩니다.

2. **모바일에서 AI 어시스턴트를 사용하려면?**
   하단의 AI 어시스턴트 버튼을 탭하면 전체 화면으로 열립니다.

3. **휴대폰으로 서류를 사진 찍어서 바로 올릴 수 있나요?**
   네, 파일 업로드를 탭하면 카메라 촬영 또는 갤러리 선택이 가능합니다. 촬영한 사진은 자동으로 글자 인식 처리됩니다.

## AI 어시스턴트

1. **AI 어시스턴트로 무엇을 할 수 있나요?**
   고객 등록/수정/삭제, 문서 관리, 계약 정보 조회, 메모 추가 등 다양한 작업을 자연어로 요청할 수 있습니다.

## 용어 설명

1. **AR(Annual Report)이란 무엇인가요?**
   보험사가 매년 피보험자에게 발송하는 연례보고서입니다.

2. **활성 고객과 휴면 고객의 차이는 무엇인가요?**
   활성 고객은 현재 관리 중, 휴면 고객은 기본 목록에서 숨겨지지만 삭제되지 않으며 언제든 복원 가능합니다.
