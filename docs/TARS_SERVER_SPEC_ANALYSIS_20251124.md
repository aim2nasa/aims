## 📊 현재 서버 스펙

### 하드웨어 구성

```
CPU:      4코어
RAM:      7.7GB (사용 4.2GB, 가용 3.5GB)
스토리지:
  - 시스템 파티션 (/dev/sda2): 116GB (76GB 사용, 35GB 남음, 69%)
  - 데이터 파티션 (/dev/sdb1): 1.8TB (100GB 사용, 1.7TB 남음, 6%)
```

### 운영 중인 서비스

- **MongoDB**: 고객 및 문서 데이터베이스 (포트 27017)
- **Node.js API 서버**: AIMS API (포트 3010)
- **Python FastAPI 서버** (3개):
  - Document Status API (포트 8000)
  - RAG API (포트 8000, host network mode)
  - Annual Report API (포트 8004)
- **Qdrant**: 벡터 데이터베이스 (문서 검색)
- **n8n**: 워크플로우 자동화
- **Docker**: 컨테이너 관리


**경고 기준:**
- 가용 RAM < 1GB → 즉시 업그레이드 필요

### 2. 디스크 사용량

**모니터링 명령:**
```bash
# 디스크 사용량 추이
df -h /data

# MongoDB 데이터 크기
du -sh /data/mongodb

# 전체 데이터 디렉토리 분석
du -sh /data/*
```

**경고 기준:**
- /data 사용률 > 80% (현재 6%)
- 시스템 파티션 > 90% (현재 69%)

### 3. CPU 부하

