# ISSUE-1: 업로드 프로그레스바 미표시

> **발견일**: 2026-03-08
> **심각도**: Minor (기능 동작에 영향 없음, UX 이슈)
> **연관 작업**: 캐치업코리아 446건 v4 분류 체계 튜닝 샘플 업로드
> **상태**: 미해결

---

## 현상

- 446건 대량 업로드 시 처리 진행 상황을 보여주는 프로그레스바가 전체 문서 보기 화면에 표시되지 않음
- 개별 문서의 처리 상태(70%, 100%, 완료)는 표시되지만, 전체 배치 업로드의 총 진행률을 한눈에 볼 수 없음

## 영향

- 사용자가 전체 처리 진행 상황을 한눈에 파악 불가
- 대량 업로드 시 "지금 몇 개까지 처리했는지" 확인하려면 목록을 스크롤해야 함

## 재현 조건

- 고객별 문서함에서 대량 파일(100건+) 업로드
- 전체 문서 보기 화면에서 진행 상황 확인

## 예상 원인

- 프로그레스바 컴포넌트가 currentBatchId 기반으로 동작하는데, 대량 업로드 시 배치 추적이 제대로 되지 않을 수 있음
- 또는 전체 문서 보기(documents-library)에서는 배치 프로그레스바가 표시되지 않는 구조일 수 있음

## 해결 방향

- 프로그레스바 컴포넌트의 배치 추적 로직 확인
- 전체 문서 보기에서도 현재 처리 중인 배치 진행률 표시
