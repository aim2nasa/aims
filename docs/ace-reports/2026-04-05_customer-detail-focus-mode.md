# ACE Report: 고객 상세 페이지 Focus Mode

## 개요
- 날짜: 2026-04-05
- 브랜치: feat/customer-detail-focus-mode
- 크기: L (Large)
- 목적: 고객 상세 페이지(CustomerFullDetailView)에 Focus Mode 추가. 2x2 그리드 외에 각 섹션을 전체 너비로 크게 볼 수 있는 레이아웃 모드.

- Dana UX 제안: Focus Mode (섹션별 풀스크린 전환 + 탭 네비게이션)
- 핵심 가치: 계약 테이블(5~7행→15~20행), 문서 리스트(파일명 잘림 해소)
